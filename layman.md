probesite is a monitoring tool that automatically checks whether your websites and servers are up and responding correctly. You point it at a list of URLs or network ports, and it tests each one — verifying the page loads, contains expected content, and responds fast enough — then reports the results in a simple table or exports them as metrics your monitoring stack can ingest. It's built for developers and system administrators who need a lightweight, self-hosted way to run synthetic uptime checks from the command line or a CI pipeline, without signing up for any third-party service.
